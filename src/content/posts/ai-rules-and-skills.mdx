> 本文汇总 **个人规则**、**项目约定（zfsoft admin-web）** 与 **通用技能（前端设计 / 代码生成）**，可直接复制到团队文档或 AI 规则配置中使用。

## 一、个人规则

### 通用编码

- 所有回答都使用中文表述。
- 生成代码时添加函数级别的中文注释。

### 代码可读性

- 遵循项目或语言的命名规范。
- 使用有意义的、描述性的名称。
- 避免缩写和单字母变量。
- 避免多层嵌套，请提前返回。

### 性能优化

- 避免不必要的对象创建。
- 及时释放不再需要的资源。
- 注意内存泄漏和线程安全问题。
- 使用适当的数据结构和算法。

## 三、通用技能

### 3.1 前端设计（`frontend-design`）

**适用场景**：原型设计、页面样式与视觉优化。

**技能元数据**：

```yaml
name: frontend-design
description: 创建独具特色、具备专业生产水准且设计质量上乘的前端界面。当用户要求构建网页组件、页面、海报或应用程序（例如网站、登录页面、仪表盘、React 组件、HTML/CSS 布局，或者对任何网页用户界面进行样式美化/美化处理）时，请使用此技能。该技能能够生成富有创意、精致的代码和用户界面设计，避免采用通用的 AI 设计风格。
```

这项技能用于产出**有特色、可上线**的前端界面，避免常见的「AI 粗糙感」。应对美学细节与创意选择保持高标准，并输出可落地的代码。

**输入说明**：用户会描述要构建的组件、页面或应用，以及用途、受众、技术限制等。

#### 设计思维

在开始编码之前，先要理解整体背景，并确定一个鲜明的视觉风格方向：
- 目的：这个界面旨在解决什么问题？谁会使用它？
- 风格：选择一种极端的风格：极度简约、极繁混乱、复古未来主义、有机/自然、奢华/精致、趣味性/玩具感、编辑/杂志风格、粗犷/原始、艺术装饰风格/几何、柔和/淡色、工业/实用主义等等。有如此多的风格可供选择。利用这些作为灵感来源，但要设计出符合该视觉风格方向的作品。
- 限制条件：技术要求（框架、性能、可访问性）。
- 差异化：是什么让这个界面令人难忘？是什么会让某人记住它？

**关键提示**：先确立清晰的概念方向，再精准落地。极致繁复与极简都可以，关键是**意图明确**，而非一味堆效果。

随后实现可运行代码（HTML/CSS/JS、React、Vue 等），并满足：
- 具有生产级的稳定性和功能性
- 视觉上引人注目且令人难忘
- 与明确的美学观点保持一致
- 在每一个细节上都经过精心打磨

#### 前端美学指南

- 字体设计：选择美观、独特且引人注目的字体。避免使用像 Arial 和 Inter 这样的普通字体；相反，应选择具有独特性的选择，以提升前端的美感；采用出人意料、富有个性的字体选择。将独特的显示字体与精致的正文字体搭配使用。
- 色彩与主题：坚持一致的风格。使用 CSS 变量以保持一致性。具有鲜明对比的主色调搭配强烈的强调效果，比平淡、均匀分布的配色方案更出色。
- 动态效果：使用动画来实现效果和微交互。优先使用纯 CSS 解决方案来处理 HTML。在可用的情况下，使用 Motion 库来处理 React。专注于高影响力时刻：一次精心编排的页面加载，通过交错展示（动画延迟）营造出更多的愉悦感，比零散的微交互效果更好。使用滚动触发和会触发悬停状态的元素来制造惊喜。
- 空间布局：出人意料的布局。不对称性。重叠。对角线流动。打破网格的元素。充裕的负空间或可控的密度。
- 背景与视觉细节：营造氛围和深度，而非默认使用纯色。添加与整体美学相匹配的背景效果和纹理。应用诸如渐变网格、噪波纹理、几何图案、分层透明度、强烈阴影、装饰边框、自定义指针以及纹理叠加等富有创意的元素。

> **避免**：滥用 Inter、Roboto、Arial 等「万能字体」；白底紫渐变等陈旧配色；千篇一律的组件骨架；与场景无关的套模板设计。  
> 鼓励针对具体场景做非常规但合理的取舍；同一站点内也应在字体、色板、气质上保持**有意识的**变化，而不是所有页面同一种「太空粗体」套路。

**重要提示**：实现复杂度要与美学目标匹配——极繁需要更丰富的动画与层次；极简则依赖留白、字号与微细节的精确控制。

### 3.2 前端代码生成（`frontend-codegen`）

**适用场景**：常见 Vue 3 项目中的 CRUD 页面脚手架。

**技能元数据**：

```yaml
name: frontend-codegen
description: 前端代码生成
```

本工具用于快速生成前端 CRUD，一般包含：

1. **API 接口文件**：与后端交互。
2. **页面组件**：列表、分页、搜索、增删改、启禁用、导出等。

#### 技术栈
- 核心：Vue3/Vue Router
- UI框架：Element Plus/Remixicon/TailwindCSS
- 工具：Axios / Vite 7

#### 开发流程
- 分析需求和接口文档
- 生成API接口代码
- 生成页面组件代码
- 配置路由
- 编写 Playwright 测试脚本

#### 开发规范
- 前端代码位于`frontend`目录下
- 页面组件命名采用小写，如`index.vue`/`list.vue`
- 变量和函数命名采用小驼峰命名法，如`getUserList`
- 常量命名采用全大写加下划线，如`API_BASE_URL`
- 每个文件都需要添加注释，模板如下：

  ```html
  <!--
  @description 文件描述
  @author zfsoft
  @date 日期格式 yyyy-MM-dd HH:mm
  -->
  ```

#### 生成文件结构
```text
src/
├── api/              # API 接口定义目录
│   └── example.js    # 模块 API
└── views/            # 页面组件目录
    └── example/      # 模块目录
```

#### API 接口定义

文件路径：`src/api/example.js`
 
```javascript
import request from '@/utils/request'
 
/**
 * 获取示例分页列表
 * @param {Object} params 查询参数
 * @returns {Promise} 请求结果
 */
export function getExampleList(params) {
  return request({
    url: '/web/sys/example/page',
    method: 'get',
    params
  })
}
 
/**
 * 获取示例详情
 * @param {String} oid 主键ID
 * @returns {Promise} 请求结果
 */
export function getExampleDetail(oid) {
  return request({
    url: '/web/sys/example/view',
    method: 'get',
    params: { oid }
  })
}
 
/**
 * 保存示例
 * @param {Object} data 示例数据
 * @returns {Promise} 请求结果
 */
export function saveExample(data) {
  return request({
    url: '/web/sys/example/save',
    method: 'post',
    data
  })
}
 
/**
 * 删除示例
 * @param {String} oid 主键ID
 * @returns {Promise} 请求结果
 */
export function deleteExample(oid) {
  return request({
    url: '/web/sys/example/delete',
    method: 'post',
    params: { oid }
  })
}
 
/**
 * 启用/禁用示例
 * @param {String} oid 主键ID
 * @returns {Promise} 请求结果
 */
export function ableExample(oid) {
  return request({
    url: '/web/sys/example/able',
    method: 'post',
    params: { oid }
  })
}
```

```vue
const {proxy} = getCurrentInstance()
const tableList = ref([])

const loading = ref(true)
const showSearch = ref(true)
const total = ref(0)
const title = ref('')
const ids = ref([])
const single = ref(true)
const multiple = ref(true)
 
const data = reactive({
  form: {
    field1: '',
    field2: '',
  },
  queryParams: {
    pageNumber: 1,
    pageSize: 10,
    field1: '',
    field2: '',
    // todo 根据需求添加其他搜索条件
  },
  rules: {

  }

/**

  } else {

  }
  loading.value = false
}
 
/**

 */
const handleQuery = () => {

  getList()
}

  resetForm()

  single.value = selection.length !== 1
  multiple.value = !selection.length

 * 新增

  resetForm()
  dialogVisible.value = true
  title.value = '新增'

        dialogVisible.value = false

        ElMessage.success(response.message)

        ElMessage.error(response.message)

    ElMessage.error(response.message)

    cancelButtonText: '取消',

    getList()
    ElMessage.success(response.message)
  } else {
    ElMessage.error(response.message)
  }
}

// 初始化
onMounted(() => {
  getList()
```
